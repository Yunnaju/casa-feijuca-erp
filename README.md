# 📖 Sobre o Projeto

O **Casa Feijuca ERP** é um sistema completo desenvolvido para gerenciamento operacional de restaurantes.

O projeto foi criado com foco em simular um ambiente empresarial real, integrando todos os setores do restaurante em uma única aplicação.

Entre as funcionalidades estão:

- Controle de usuários
- Controle de permissões
- Gestão de estoque
- Cadastro de produtos
- Cadastro de fornecedores
- Atendimento por comandas
- Controle de mesas
- Lançamento de pedidos
- Painel do Bar
- Controle do Caixa
- Registro de pagamentos
- Fechamento de Caixa
- Dashboard Operacional
- Histórico de movimentações
