# 🍕 Casa Feijuca ERP

Sistema de gestão para restaurantes desenvolvido com HTML, CSS e JavaScript.

## Sobre o projeto

O Casa Feijuca ERP foi desenvolvido para digitalizar a operação de um restaurante, integrando atendimento, comandas, estoque, caixa e controle de usuários.

## Funcionalidades

- Login com controle de acesso por perfil
- Perfis de Administrador, Gerente, Caixa, Garçom e Bar
- Abertura e fechamento de comandas
- Lançamento de bebidas e pizzas à la carte
- Observações para pedidos de bebidas
- Cadastro de produtos e fornecedores
- Recebimento, conferência e armazenamento
- Controle de estoque, perdas e inventário
- Registro de pagamentos
- Fechamento de caixa
- Dashboard operacional
- Histórico de movimentações
- Interface responsiva para desktop e celular

## Tecnologias

- HTML5
- CSS3
- JavaScript
- LocalStorage
